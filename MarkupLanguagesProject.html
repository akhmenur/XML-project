Introduction

This project represents the preferences of an Google application for mobile. It shows the table with chosen preferences that can be chosen in this mobile application. Also this preferences ordered by Alphabet order and have several blocks marked with color. Green color for reminder frequency set on less than 10 days and red for more than 10 days.

Methodology

For making this project I used this technologies

  • XML
  • XML Schema
  • HTML
  • XSL
  • XSLT

All documenst are valid and well formed. Checked using W3C XML Schema (XSD) Validation online

Work content

This project consists of the following files:

  1. Xml file
  2. Xsd file
  3. Xslt file
  4. Html file

Elements description

The root element is Google_app_settings. It contains 5 different records named Settings. Each of them had different preferences chosen.

Each Settings contains the following:

  • search_language
  • account
  • account_nickname
  • account_mail
  • search_region
  • notification_settings
  • doodles
  • safe_search_fiter
  • open_web_pages_in_app
  • feed
  • speech_output
  • enable_recent
  • news_frequency
  • frequency
  • Theme_settings
  • font
  • theme
  • brightness
